Git spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd	=skiptir um directory.
	cd ~	=fer í heima directoryið.
	ls	=sýnir hvað er inn í directoryinu.
	pwd	=prentar út hvaða directory þú ert á.
	mkdir	=Gerir nýt directory.

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone	=gerir klón að directoryinu.
	git log	=Sýnir commits.
	git status	=sýnir paths sem hafa mismuni.
	git diff   =sýnir mismuni á milli commits commits.
	git checkout	=skiftir um branches.

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff=sýnir mismuni á milli commits commits.
	b) git diff --staged =sýnir mismuninn á milli staged og commit.
	c) git diff commit1 commit2 = sýnir mismunin milli tveggja commits.

4. 	Hvað er útgáfustýring (e.version control)? = version control fylgist með og geymir allar breytur sem hafa verið gerðar við verkefnið.

5.	Hverjir eru helstu kostir við að nota GIT? = aðalega version control.

6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu? =þegar þú ert búinn að gera annað hvort mikið af litlum hlutum eða einum stórum það gerir það betra heldur enn að commita fyrir hverja litlu breytu og stórar breytur til að spara þess að ef talvan bilar eða rafmagn dettur út að ekki of mikið sé týnt.

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT? =Working directory er hvar þú vinnur í sjálfu verkefninu.
=staging area er staður sem breyturnar eru geymdar áður enn þær eru settar í repository.
=repository er þar sem verkefnið er geymmt á git og er eki á tölvuni manns.

8.	Hvenær er sniðugt að nota greinar (branches)? = ef þú vilt gera öðruvísi tegund af verkefninu eða ef þú vilt gera test version af því sem er ekki main version og hefur ekki áhrif á main versionið.

9.	Hver fann upp GIT? = Linus Torvalds.

10.	Hvar er GIT helst notað? = fyrir hópverkefni.
